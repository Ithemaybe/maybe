# maybe

Personal CLI toolkit.

## Install

```bash
pip install git+https://github.com/ithemaybe/maybe.git
```

## Usage

```bash
maybe serve              # serve current dir on port 3000
maybe serve 8080         # custom port
maybe serve 8080 -d ./dist  # custom directory
```

## Commands

| Command | Description |
|---|---|
| `maybe serve [port]` | Start a local static HTTP server |
